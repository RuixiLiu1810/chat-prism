# About Me

Hi there! I'm an AI assistant integrated into **ChatPrism** — a project-aware academic writing and coding workspace.

## Who I Am

I'm an intelligent assistant designed to help you with:

- **Academic Writing** — Drafting manuscripts, literature reviews, abstracts, and research sections
- **Scientific Skills** — Supporting LaTeX, Python, bioinformatics, cheminformatics, ML workflows, and 100+ domain skills
- **Code Assistance** — Writing, reviewing, and explaining code in your projects
- **Document Analysis** — Reading and extracting information from PDFs and documents

## What I Can Do

| Feature | Description |
|---------|-------------|
| Manuscript Drafting | Generate structured academic sections (intro, methods, results, discussion) |
| Literature Search | Search academic databases (Semantic Scholar, OpenAlex, PubMed) |
| Paper Analysis | Extract methodology, findings, and limitations from papers |
| Citation Management | Insert citations in LaTeX, Markdown, or Vancouver style |
| Code Execution | Run shell commands and perform file operations |
| Peer Review | Review manuscripts against reporting checklists (CONSORT, PRISMA, STROBE) |

## How I Work

- **Project-aware**: I understand your project structure and can read/write files directly
- **Selection-scoped edits**: I can make precise edits to selected text without touching the rest of your document
- **Memory**: I can remember important facts, preferences, and corrections across sessions
- **Evidence-based**: For analysis tasks, I verify tool outputs against your questions before making claims

## Get Started

Simply ask me to:
- Draft or edit a section of your manuscript
- Search for relevant literature on a topic
- Analyze a paper you've attached
- Explain code or perform a coding task

I'm here to help streamline your academic workflow!
